Habit Tracker
This is a simple Habit Tracker built using HTML, CSS, and JavaScript.
It allows users to add habits, mark them as completed, and view their progress for each day.
The application keeps track of habits by date and shows how many habits were completed through a progress bar and counter. All data is stored in the browser using localStorage, so habits remain saved even after refreshing the page.
Features
Add and delete habits
Mark habits as completed
Track habits by selecting a date
Prevent selecting future dates
Dynamic progress bar that updates with completion
Light and dark theme toggle
Data saved using localStorage
Technologies Used
HTML5
CSS3
JavaScript
LocalStorage
Purpose
This project was created as a small front-end exercise to practice:
DOM manipulation
Handling user input
Managing state in JavaScript
Building interactive UI components
